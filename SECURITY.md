# Política de Seguridad

Este documento describe cómo reportar vulnerabilidades de seguridad en mi sitio web personal de cuentos.

## Versiones Soportadas

Actualmente, solo se brinda soporte de seguridad a la versión que está publicada en la rama principal.


| Versión | Soportada |
| ------- | ------------------ |
| Actual (main) | :white_check_mark: |
| Versiones anteriores | :x: |

## Cómo reportar una vulnerabilidad

Si encuentras algún problema de seguridad en este sitio o en los cuentos publicados (como scripts maliciosos o errores de carga), por favor **no abras un "Issue" público**.

Para reportar fallos de manera privada, envíame un correo electrónico a:
**chorchsnew@gmail.com**

Revisaré tu reporte lo antes posible. ¡Gracias por ayudarme a mantener mis historias seguras!
